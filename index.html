Full responsive

El diseño web adaptable, adaptativo o responsivo, conocido por las siglas RWD,es una filosofía de diseño y desarrollo cuyo objetivo es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visualizarla.
